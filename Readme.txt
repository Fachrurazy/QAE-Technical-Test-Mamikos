How to run Project

-Pertama ubah path Driver disesuaikan dengan letak driver anda
-Ubah path driver pada isi dari TenantSearch.java dan Owner.java di method Open Browser, ini untuk membaca path letak dari Webdriver tersebut
-Lalu buka file runner.java yang berada di QAE-Technical-Test-Mamikos\src\test\java\runners\runner.java
-Melakukan Run pada runner.java

How to see Report testing
-Harus sudah melakukan run runner.java
-lalu buka folder targer->cucumber-report->report2.html
-Open with browser
